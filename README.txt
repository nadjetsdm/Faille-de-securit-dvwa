DataCleaner : Solution de Qualité de Données Open Source

DataCleaner est un ensemble d'outils de qualité de données qui vous permet de profiler, corriger et enrichir vos données. Il est utilisé pour des analyses ponctuelles, des nettoyages récurrents, ainsi que comme un couteau suisse dans les solutions de correspondance et de gestion de données de référence.
Fonctionnalités Clés
Profilage de Données : Comprendre la qualité, la structure et les incohérences de vos données.
Correction de Données : Corriger facilement les anomalies et incohérences.
Enrichissement de Données : Améliorer vos données avec des informations supplémentaires.
